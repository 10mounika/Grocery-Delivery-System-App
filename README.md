# Grocery-Shopping-System-APP
Grocery Shopping System consist of Admin and users: (Using online Firebase Database) Admin can login to the system. He also can to Add, Edit, Delete an Existence category/s  or create new category (Category name, Category cost, available Category amount, Expiry date)  Users can to register or login to the system  the system shows the existence categories (fruits, meats, Vegetables, etc...) the user can search about product and know product info (cost, available amount, Expiry date) the user also can to make an order by giving the system (Destination, product name, product amount) the delivery task is deliver this product to the user in selected time
